◆ ColorControlCommand

enum Microsoft.Azure.Kinect.Sensor.ColorControlCommand
strong

Color sensor control commands.

Enumerator
ExposureTimeAbsolute 

Exposure time setting.

May be set to Auto or Manual.

Exposure time is measured in microseconds.

Brightness 

Brightness setting.

May only be set to Manual.

The valid range is 0 to 255. The default value is 128.

Contrast 

Contrast setting.

May only be set to Manual.

Saturation 

Saturation setting.

May only be set to Manual.

Sharpness 

Sharpness setting.

May only be set to Manual.

Whitebalance 

White balance setting.

May be set to Auto or Manual.

The unit is degrees Kelvin. The setting must be set to a value evenly divisible by 10 degrees.

BacklightCompensation 

Back-light compensation setting.

May only be set to Manual.

Value of 0 means back-light compensation is disabled. Value of 1 means back-light compensation is enabled.

Gain 

Gain setting.

May only be set to Manual.

PowerlineFrequency 

Power-line frequency setting.

May only be set to Manual.

Value of 1 sets the power-line compensation to 50 Hz. Value of 2 sets the power-line compensation to 60 Hz.
